## Git Configuration

This repo belongs to the `surajtripathi` GitHub account (personal, separate from `iAmSurajT`/`stockx`).

**SSH alias**: `github.com-personal`
**Remote**: `git@github.com-personal:surajtripathi/surajtripathi.github.io.git`
**Default branch**: `master`

When pushing/pulling, the custom SSH config (`~/.ssh/config`) maps `github.com-personal` to `github.com` with the correct SSH key for this account.

Clone command pattern:
```bash
git clone git@github.com-personal:surajtripathi/<repo>.git
```

All git operations (push, pull, fetch) will automatically use this alias since it's set in the remote URL.
